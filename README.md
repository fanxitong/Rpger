# Rpger

基于[Avalonia](https://www.avaloniaui.net/)实现的类 RPG Maker 系列游戏编辑工具。  
使用最新的 .net LTS 版本开发。

## 开发环境

-   Windows 11
-   Visual Studio 2022 Community
-   .net 8

## 注意

-   此项目为我的毕业设计,并不保证会持续更新.
-   仅支持 c#
-   目前仅支持桌面环境(Windows、MacOS、Linux)
